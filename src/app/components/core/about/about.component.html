O Projektu

Projekat iz predmeta: Razvoj viseslojnih aplikacija.

Tema projekta simulira rad sistema za evidenciju podataka na nivou katedre.

Projekat je radjen u tehnologijama baziranim na jezicima Java i Java Script.
Kao baza koristi se PostgreSQL, server je napisan u Spring Boot-u, a klintska strana je kreirana uz pomoc Angular JS-a
